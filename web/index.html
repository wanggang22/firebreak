■ FIREBREAK the line the fire can't cross
● Arc testnet · verified ↗

A non-custodial liquidation firewall for stablecoin-native lending on Arc. A borrower signs an on-chain Mandate (health trigger, spend cap, action whitelist); an autonomous keeper watches the loan and, before liquidators can move, executes the cheapest bounded rescue in one atomic transaction. Below is a replay of a real rescue executed on Arc testnet — every number and the transaction are on-chain and verifiable. Claude ranks the vetted paths; sizing, spend cap, and the action whitelist stay deterministic, so the model can only pick a bounds-checked path.
